# 项目总览

PromptShip 是一个 AI 驱动的低代码 Web 应用生成平台。用户用自然语言描述需求，系统通过 AI 生成前端应用代码，支持预览、部署、下载源码和对话式继续修改。

## 1. 核心能力

- AI 智能路由：自动选择 `html`、`multi_file`、`vue_app` 三种生成类型。
- 代码生成：支持单文件 HTML、多文件 HTML/CSS/JS、Vue 工程化项目。
- 对话式迭代：应用创建后可以继续对话修改，后端保存对话历史。
- 应用管理：支持我的应用、精选案例、部署、下载、删除、重命名等能力。
- 静态预览：生成结果通过 `/static/{codeGenType}_{appId}/` 访问。

## 2. 技术栈

后端：

- Spring Boot 3.x
- JDK 21
- MyBatis-Flex
- LangChain4j
- Reactor `Flux`
- Spring MVC `SseEmitter`
- MySQL
- Hutool、Lombok、Knife4j/springdoc

前端：

- Vue 3
- Vite
- TypeScript
- vue-router
- Pinia
- Ant Design Vue
- Axios
- markdown-it
- DOMPurify
- @umijs/openapi

## 3. 关键目录

```txt
src/main/java/com/xiaoyu/promptship/
  controller/      REST 接口层
  service/         业务接口
  service/impl/    业务实现
  core/            AI 代码生成核心流程
  ai/              LangChain4j AI 接口和 Agent
  model/           DTO、VO、实体、枚举

prompt-ship-frontend/
  src/api/generated/  OpenAPI 机器生成接口
  src/api/services/   手写业务 API 封装
  src/pages/          页面
  src/components/     组件
  src/stores/         Pinia store

docs/              项目文档
tmp/code_output/   AI 生成代码输出目录
tmp/code_deploy/   部署后的静态文件目录
```

## 4. 高层链路

```txt
用户输入 Prompt
-> 后端创建 App
-> Router Agent 选择生成类型
-> 对应生成器生成代码
-> 保存代码到 tmp/code_output
-> 若为 Vue 工程化项目，执行依赖处理与 npm 构建
-> 前端通过 SSE 展示生成进度
-> 用户预览、部署、下载或继续对话
```
